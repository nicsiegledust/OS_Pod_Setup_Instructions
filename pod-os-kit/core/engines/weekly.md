# Weekly Engine (Full; optional in Standard)

Weekend, user's chosen time:

1. Weekly reset of `dashboard_data.json` and priorities surfaces; write `state/weekly/{date}.json`.
2. Week preview Slack post: the week's meetings tiered by importance, targets vs progress, top risks.
3. Deep review of everything in flight (the role pack defines what "in flight" means: deals, campaigns, projects).
4. Full-week meeting prep, tiered (deep prep for high-stakes, light for the rest).
5. Monthly, first run of the month: a Kaizen pass proposing deletions. A system that only grows is a system rotting.
6. Audit all engines: schedule.json vs live wake-ups vs handoff-status. Alert on drift.
7. Run `wakeup-relay.md`.
