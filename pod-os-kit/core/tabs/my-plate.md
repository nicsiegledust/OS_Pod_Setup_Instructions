# Tab: My Plate
Shows: every open Pod task, the single commitment ledger. Sorted P0-P3 then date.
Feeds: Pod tasks API (not a file). Refreshed: live. Janitored by: Evening.
Row: status dot, P-chip, explicit date, `{Context}: {action}` title.
Actions: Caret, Edit, Start (task_engage), Snooze (task_defer), Done (task_done). No Kill: tasks are closed or re-dated, never silently dropped.
Rules: task cap badge when over cap; `missing date` renders as a yellow warning, not a fabricated date.
