# Tab: Rituals
Shows: recurring personal/role routines (pipeline hour, content block, weekly review, gym, whatever the user declares) with streak dots.
Feeds: `me/rituals.json` (definitions) + `state/rituals-log.jsonl` (stamps via ritual_done).
Row: name, cadence chip, trailing 10-day dots.
Actions: Caret, Edit, Done. No Snooze (a ritual missed is a miss, honestly rendered).
