# Tab: Waiting On

Feeds: `state/waiting.jsonl`. Written by: Morning (detected from Sent + threads) and capture triage. Nudge dates via `waiting_update`.
Row chips: who, waiting since (relative), nudge-after date.
Actions: Caret, Edit, Start (draft the nudge; never auto-send), Snooze (push nudge date), Done, Kill.
