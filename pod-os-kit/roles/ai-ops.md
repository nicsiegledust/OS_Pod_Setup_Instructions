# Role Pack: AI Ops (agent/adoption owner)
Tabs: Today · Agents · Adoption · My Plate · Inbox.
System of record: the AI platform's usage data (warehouse or analytics API) + project tool for rollout work.
work/: `rollouts/`, `feedback/`, `meetings/`, `notes/`.
Agents tab: row per watched agent/workflow: weekly active users, run trend arrow, error/complaint count, owner. Adoption tab: teams ranked by usage trend, champions surfaced, silent teams flagged.
Watch topics (preselected): usage drops per team / new power users & champions / recurring agent failures or complaints / feedback backlog / rollout milestones / new platform features worth shipping internally.
Engine tweaks: Morning flags anomalies (usage cliff, error spike) with a Start that spawns an investigation conversation; Weekly produces the adoption report (shipped, adopted, stuck) and a champions-to-thank list.
Hard rules: feedback items become drafts for the feedback board, never auto-filed; usage numbers come from the mapped IDs, never name matching.
