# Role Pack: Ops / Project Manager
Tabs: Today · Projects · My Plate · Waiting On · Inbox.
System of record: project tool (Linear/Asana/Jira/Notion). Store board/team IDs; freeze status-field IDs from tool metadata once.
work/: `projects/`, `meetings/`, `decisions/`, `notes/`.
Projects tab: row per active project: stage chip, next milestone date, owner, top blocker. Expanded: RAID snapshot (risks, assumptions, issues, dependencies).
Waiting On is first-class here: most PM pain is unanswered asks. Morning detects overdue dependencies and stale approvals; nudge drafts, never auto-sent.
Watch topics (preselected): milestone slippage / blocked threads & stale approvals / meetings & prep / decision log gaps / stakeholder silence >N days.
Engine tweaks: Morning leads with what slipped overnight and today's decision points; Evening logs decisions made (append work/decisions/); Weekly produces the portfolio review (on-track/at-risk/late) and next week's critical path.
